Open Ready-for

Instructions

  1. Set an username and password (or leave default) and generate QR code.
  2. Detect your phone local IP address. Phone and PC must be connected on the same network.
  3. Open your RDP client (ex. Remmina on Linux or Windows Remote Desktop Client) and create a new connection using your phone address and username and password chosen. It's important that credentials are set on client before connection.
  4. Scan the QR code with your phone and wait until a message saying Connecting to PC appears.
  5. From your RDP client start the connection to the phone.
